# 常见的一些运算符


1. `>>` 右移运算符 `num >> 1 ` 相当于 `Math.floor(num/2)`
2. `<<` 左移运算符 `num<<1` 相当于`Math.floor(num) * 2`
3. `~`  按位取反 一般用于处理 -1 的情况, 因为~-1 是 0 所以省去了判断 > -1;
4. `~~` 取反再取反 相当于调用了`Math.floor(num)`
5. `!!` 转化为布尔值 配合`~`使用
6. `**` 幂 等价于 Math.pow();
7. `?.` 可选链
8. `??` 判断undefined 或者是 null
